# ChartTab

Motivational charts on the Chrome's new tab page.

## Build & development

Run `grunt` for building and `grunt debug` for preview.

## Unit tests

Run `npm install` and then `npm run unit` to execute the test suite. Use `npm test` to run ESLint.
